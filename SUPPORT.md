# Support

GitHub issues are for reproducible lab defects, documentation clarity issues,
and bounded feature proposals about the portable evidence lab.

This project is a personal reference project and does not provide production support,
managed SRE support, or emergency incident response.

Security issues must use GitHub private vulnerability reporting. Do not open a
public issue with exploit details, credentials, tokens, private customer data,
or production secrets.

Please include the exact validation command, evidence file path, or local log
needed to reproduce the issue. The best-effort public triage target is 7 days,
but this target is not a service-level agreement.

Requests that require paid services, external credentials, organization-specific
endpoints, or private production data are outside the support boundary.
